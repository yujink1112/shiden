# 主要人物視点で見た物語

リザベルと同じように、主要人物ごとに  
`その人の視点で見ると、この物語がどう見えるか`  
を整理したメモです。

対象は以下の5人です。

- レミエル
- ルーサー
- アダム
- ラスト・ウィッチ号
- エーヴィス

## レミエル視点

レミエル視点で見ると、この物語は  
`子供の頃に抱いた夢を、現実の汚さも痛みも全部抱えたまま、それでも諦めずに追い続ける話`  
です。

彼は一見すると、最初から何でもできる主人公に見えます。

- 発想力がある
- 度胸がある
- 人を惹きつける
- ちゃんと勝つ

でも内側にあるのは、かなり強い `執着` と `後悔` です。

《アノマ》を目指すのは、単にロマンがあるからではなく、  
幼い頃の夢を絶対に裏切りたくないからです。

しかもその夢には、エーヴィスとの記憶がこびりついている。  
つまりレミエルにとって冒険は、

- 希望
- 自己実現
- 贖罪

が全部混ざったものになっています。

だから彼は、軽口を叩きながらも本気で危険に踏み込める。  
夢を叶えたいだけでなく、途中で止まることが「昔の自分をまた裏切ること」になるからです。

エーヴィスとの再会以降、この物語はレミエル視点では  
`夢を叶える旅` であると同時に  
`過去に置き去りにした親友と向き合う旅`  
にもなります。

リザベルを助ける判断も同じです。  
合理的ではないし、危険でもある。  
それでも見捨てられないのは、彼の海賊観が

`夢を見ること`

`弱い者を見捨てないこと`

と結びついているからです。

だからレミエル視点の物語は、  
`夢を叶えることと、人間であり続けることの両立を試され続ける話`  
だと言えます。

## ルーサー視点

ルーサー視点で見ると、この物語は  
`自分に自信のなかった青年が、誰かを守ることで自分の居場所を見つけていく話`  
に見えます。

ルーサーは最初から強いです。  
でも本人には、あまり「自分はすごい」という感覚がない。

- 真面目
- 力持ち
- 面倒見がいい
- 正確で実直

という長所があるのに、それを当たり前だと思っていて、自分の価値として強く自覚していない。

だから彼の軸になるのは、いつも `誰かの役に立てるか` です。

レミエルに誘われたこと、海軍で挫折したこと、海賊団に入ったこと。  
その全部が、ルーサーの中では  
`自分でも居ていい場所を探す過程`  
になっています。

リザベルに対して強く肩入れするのも、ルーサーらしいです。  
彼は「守るべき相手」を前にすると一気に主体性を持つ。  
だからリザベルの存在は、ルーサー視点だと

- 優しさを向ける対象
- 自分の正しさを試す相手
- 裏切りによって傷つけられる相手

の全部になっています。

12章でリザベルの正体が判明した時、  
ルーサー視点では「敵だった」という事実以上に、  
`信じたかった気持ちが裏切られた`  
ことの方が大きいはずです。

それでも最後に彼が完全に彼女を切り捨てきれないのは、  
ルーサーという人物の良さが、正しさより先に `情` にあるからです。

つまりルーサー視点の物語は、  
`力を持つこと` ではなく  
`優しさを持ったまま強くなること`  
の話として読めます。

## アダム視点

アダム視点で見ると、この物語は  
`すべてを失った老人が、若者たちの無謀な夢にもう一度賭け直す話`  
です。

アダムはパーティの中で最も達観して見えます。  
けれど、彼はもともと伝説級の海賊であり、既に一度すべてを喪失した人間です。

- 仲間を失った
- 船を失った
- 名も過去も捨てた

そのうえでレミエルに拾われている。  
だから彼にとって《アノマ》への旅は、

- 若者の夢への付き添い
- 自分が終わった人間ではないという再確認
- 亡き仲間たちへの弔い

を兼ねています。

アダムはしばしば冷静で、理性的で、抑制的です。  
でもその内側には、まだ熱が残っている。  
ゴルゾアの胃袋や深海での場面では、それがかなりはっきり見えます。

彼は若者たちよりも死を近く感じている。  
だからこそ、彼らの無謀さに呆れながらも、本気で支えようとする。  
自分がもう夢を見ない代わりに、夢を見る若者たちに最後の剣を貸している感じです。

エーヴィスやリザベルのような若さの暴走を見ても、  
アダムは比較的すぐに切り捨てず、  
どこか一歩引いて「そういうものだ」と見ている。  
それは年長者の余裕でもあり、自分にもかつてそういう熱があった記憶の裏返しでもあるでしょう。

アダム視点の物語は、  
`人生の終わりかけにもう一度だけ、誰かの夢を信じてみる話`  
として読むと、とても切ないです。

## ラスト・ウィッチ号視点

ウィッチ視点で見ると、この物語は  
`長い停滞の末に再び世界へ引きずり出され、嫌々ながらもう一度生を始める話`  
です。

彼女は最初、かなり醒めています。

- 人間を信用していない
- 期待していない
- 契約も半ば打算

これは単なる皮肉屋というより、長い時間と喪失がそうさせています。  
魔女としての歴史、滅び、生き延びるために船の身体を得たこと。  
そうした過去を踏まえると、彼女にとって世界はもう  
`関わるだけ損をする場所`  
に近い。

だからレミエル達との旅は、ウィッチ視点では最初から希望ではありません。  
むしろ

- 面倒
- 騒がしい
- 非合理

の連続です。

でも、その面倒さの中で少しずつ変わっていく。  
レミエルの無茶、ルーサーの素直さ、アダムの老獪さ、リザベルの危うさ。  
そうした人間達を観測していくうちに、彼女は少しずつ `当事者` になっていきます。

重要なのは、ウィッチが感情豊かになるというより、  
`無関心ではいられなくなる` ことです。

彼女にとっての変化は、

- 人間を好きになること

ではなく、

- 人間の運命に自分も巻き込まれてしまうこと

に近い。

リザベルに対して厳しいのも、単なる理性だけではないでしょう。  
同じ魔女だからこそ、未熟さの危険も、孤独の歪みもよく分かっている。  
だからあの扱いは処刑でも擁護でもなく、かなり彼女らしい距離感になっています。

ウィッチ視点の物語は、  
`凍っていた時間が、他人との航海によって少しずつ動き出す話`  
として読むのがいちばんしっくりきます。

## エーヴィス視点

エーヴィス視点で見ると、この物語は  
`奪われた少年が、裏切られた記憶を燃料にして、自分だけの夢を海賊の形で肥大化させていく話`  
です。

彼にとってレミエルは、ただのライバルではありません。

- 一緒に夢を見た相手
- 自分を置いていった相手
- それでも忘れられない相手

です。

だからエーヴィスの海賊としての野心には、純粋なロマンだけでなく、  
強い怨みと欠落が混じっています。

《アノマ》を目指すのも、単に財宝や伝説のためだけではない。  
`自分が間違っていなかったと証明したい`  
し、  
`置き去りにされた側でもここまで来られると示したい`  
んです。

彼が仲間達に慕われているのも重要で、  
エーヴィスは単なる拗らせた被害者ではありません。  
ちゃんと船長として人を惹きつけ、夢を語り、団をまとめている。  
だからこそ悲劇になる。

レミエル視点だとエーヴィスは「失った親友」ですが、  
エーヴィス視点だとレミエルは  
`奪われた未来そのもの`  
です。

彼がレミエルを異様なまでに偽物扱いするのも、  
本当は「お前は俺と同じ夢を見ていたくせに、なぜそっち側に行った」という怒りがあるからでしょう。

終盤での死に際は、その怒りが最後まで消えないまま、それでも完全な憎しみにはなりきれないのが苦しい。  
指輪を渡す場面は呪いであり、遺言であり、執着であり、絆でもある。  
だからあの決着は、勝敗以上に関係の終わりとして重いです。

エーヴィス視点の物語は、  
`裏切られたと思っていた相手を、最後まで憎みきれなかった男の悲劇`  
として読むと、とても効きます。

## まとめ

同じ物語でも、視点を変えると見え方がかなり変わります。

- レミエル視点では `夢と贖罪の冒険譚`
- ルーサー視点では `優しさを持ったまま強くなる話`
- アダム視点では `終わったはずの人生にもう一度夢が差し込む話`
- ウィッチ視点では `凍っていた時間が再び動き出す話`
- エーヴィス視点では `置き去りにされた少年の悲劇`

この物語が面白いのは、  
誰か一人の物語だけで閉じず、それぞれにちゃんと別の物語が流れているからだと思います。
